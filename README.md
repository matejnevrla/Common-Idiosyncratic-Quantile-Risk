# Common-Idiosyncratic-Quantile-Risk

This repository accompanies Baruník & Nevrla (2026). In particular, it contains data on:
1. *Common Idiosyncratic Quantile* (CIQ) factors in the file `ciq_factors_2024_public_data.xlsx`.
2. Returns of the high minus low portfolios sorted by exposure to the $\Delta \textrm{CIQ}$ factors in the `portfolio_returns_2024_public_data.xlsx` file. Returns are obtained from decile sorts and both equal and value-weighted returns are reported.

Baruník, J. and Nevrla, M. (2026): *Common Idiosyncratic Quantile Factors and Asset Prices*. [Available for download here](https://matejnevrla.github.io/files/common_idio_quant_revised.pdf).
